# 竜王伝説（仮）— ブラウザRPG

ドラゴンクエスト1を題材に、ジャンルの仕組み（フィールド探索・ランダムエンカウント・
コマンド戦闘・レベル/呪文成長・装備・セーブ）を約8割模倣したオリジナル本格RPG。
固有名詞・台詞・マップ・グラフィック・音楽はすべてオリジナル新規作成。
バニラ JavaScript + Canvas2D + Web Audio、依存ライブラリゼロ。

## 遊び方
- `竜王伝説.html` をブラウザで開く（単一ファイル・ダブルクリックで動作）
- 開発用の分割版は `index.html` + `src/`（ES Module。ローカルサーバ経由で起動）

## 操作
- 移動: 方向キー / WASD（クリック/タップ可）
- 決定: Enter / Space / Z　キャンセル: Esc / X　メニュー: M / Shift
- BGMミュート: B　セーブ書き出し/読み込み（タイトル画面）: E / I

## 主な機能
- 6+1マップ（城/城下町/平原/洞窟/最深部/沼の里/竜王城/宝物庫）、敵16体、竜王2段階戦
- レベル/経験値/呪文習得、武器・防具・道具店、ぼうけんのしょ（3日有効のlocalStorage+Cookie保存）
- ファイル書き出し/読み込みによる確実なセーブ移行
- Web AudioによるオリジナルチップチューンBGM（場面連動）

## 構成
- `竜王伝説.html` … 配布用の自己完結シングルHTML（ビルド成果物）
- `index.html`, `src/` … 開発用ソース（ES Module）
- `assets/` … 画像差し替え用ディレクトリ（現在はプログラム描画のプレースホルダ）
